# Contributing to DevTree Static

Thanks for your interest in contributing! We welcome contributions of all kinds.

## How to Contribute

### Reporting Bugs
- Check if the bug has already been reported
- Include a clear description and steps to reproduce
- Include your configuration files (sanitized)
- Browser and OS information

### Suggesting Features
- Check existing issues first
- Describe the feature and use case clearly
- Explain why this would be useful
- Provide examples if possible

### Pull Requests
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test thoroughly
5. Commit with clear messages: `git commit -am 'Add feature'`
6. Push to the branch: `git push origin feature/your-feature`
7. Open a Pull Request

## Development Setup

```bash
# Clone the repository
git clone https://github.com/bedlam520Dev/devtree-static.git
cd devtree-static

# Install dependencies (optional)
npm install

# Start development server
npm run dev
# or
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Code Style

- Use consistent indentation (2 spaces)
- Keep functions small and focused
- Add comments for complex logic
- Follow existing code patterns

## Testing

- Test in multiple browsers
- Test on mobile devices
- Verify dark mode works
- Check accessibility

## Documentation

- Update README if adding features
- Add comments to complex code
- Document configuration options
- Include examples

## Questions?

Feel free to open an issue or start a discussion!
